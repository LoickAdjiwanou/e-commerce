# E-commerce back-end

**Free hosting use for deployment**


 ## Technology Used   
 - Nodejs
 - Mongodb
 - Express

<br />

 ## Step to Start Project
 
 1. `git clone https://github.com/LoickAdjiwanou/e-commerce.git`
 2. `cd e-comerce-backend`
 3. `npm i`
 4. `npm run data:import`
 5. `npm run dev`
